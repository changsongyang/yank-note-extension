# Milkdown

This extension integrates [Milkdown](https://milkdown.dev/) into Yank Note.

> **Warning**: 当前扩展处于实验阶段，可能会有不稳定的情况发生，使用需谨慎。

> **Warning**: This extension is in the experimental stage and may be unstable. Use with caution.
