# ECharts Extension

This extension integrates [ECharts](https://github.com/apache/echarts) into Yank Note.

## Usage

~~~markdown
```js
// --echarts--

const option = {
  xAxis: {
    type: "category",
    data: ["Mon", "Tue", "Wed", "Thu", "Fri", "Sat", "Sun"]
  },
  yAxis: {
    type: "value"
  },
  series: [
    {
      data: [150, 230, 224, 218, 135, 147, 260],
      type: "line"
    }
  ]
}

chart.setOption(option, true)
```
~~~
